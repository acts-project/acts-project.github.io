Detailed Description

template<std::size_t kDim = 2u, std::size_t bSize = 100u>
class ActsPlugins::Geant4SurfaceProvider< kDim, bSize >

A surface provider that extracts surfaces from a gdml file.

This provider extracts volumes from a gdml file based on the preselection criteria and converts them to surfaces. By default, all the volumes are converted.

Optionally, it can be configured to return a range-based subset of all the preselected surfaces. This is done by setting the range and binning values in the kdtOptions

Note
if the KDTree selection is not needed, the template parameters can be left to their default values as they will not affect the result.
Template Parameters
kDimThe number of dimensions for the KDTree
bSizeThe maximum number of surfaces per KDTree leaf
