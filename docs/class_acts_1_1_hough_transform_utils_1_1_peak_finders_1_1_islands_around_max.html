Detailed Description

template<class identifier_t>
class Acts::HoughTransformUtils::PeakFinders::IslandsAroundMax< identifier_t >

Peak finder inspired by ATLAS muon reconstruction.

Looks for regions above a given fraction of the global maximum hit count and connects them into islands comprising adjacent bins above the threshold. Peak positions are averaged across cells in the island, weighted by hit counts

Template Parameters
identifier_tThe identifier type to use. Should match the one used in the hough plane.
