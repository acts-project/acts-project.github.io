◆ lastMeasurementIndex

template<typename traj_t>
std::size_t Acts::KalmanFitterResult< traj_t >::lastMeasurementIndex = MultiTrajectoryTraits::kInvalid

This is the index of the 'tip' of the track stored in multitrajectory.

This corresponds to the last measurement state in the multitrajectory. Since this KF only stores one trajectory, it is unambiguous. MultiTrajectoryTraits::kInvalid is the start of a trajectory.

◆ lastTrackIndex

template<typename traj_t>
std::size_t Acts::KalmanFitterResult< traj_t >::lastTrackIndex = MultiTrajectoryTraits::kInvalid

This is the index of the 'tip' of the states stored in multitrajectory.

This corresponds to the last state in the multitrajectory. Since this KF only stores one trajectory, it is unambiguous. MultiTrajectoryTraits::kInvalid is the start of a trajectory.
