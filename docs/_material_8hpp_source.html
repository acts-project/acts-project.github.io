47 // Both mass and molar density are stored as a float and can thus not be
48 // distinguished by their types. Just changing the last element in the
49 // previously existing constructor that took five floats as input to represent
50 // molar density instead of mass density could have lead to significant
51 // confusion compared to the previous behaviour. To avoid any ambiguity,
