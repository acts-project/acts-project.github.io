48 // Cache the main grid and the density contributions (trackGrid and z-bin)
49 // for every single track.
50 // This option enables the possibility to calculate the entire main grid

52 // collection, the individual track density contributions to the main grid
53 // can just be removed without calculating the entire grid from scratch.
