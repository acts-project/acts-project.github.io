Detailed Description

template<class identifier_t>
class Acts::HoughTransformUtils::PeakFinders::LayerGuidedCombinatoric< identifier_t >

Peak finder inspired by ATLAS ITk event filter developments.

Builds peaks based on layer counts and allows for subsequent resolution of the combinatorics by building multiple candidates per peak if needed. In flat regions, peak positions are moved towards smaller values of the second and first coordinate.

Template Parameters
identifier_tThe identifier type to use. Should match the one used in the hough plane.
