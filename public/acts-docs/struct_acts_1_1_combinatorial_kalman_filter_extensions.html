Definition CombinatorialKalmanFilterExtensions.hpp:47

Delegate the extension of the trajectory onto the given surface to an external unit.

Note
Expected to create track states for measurements associated to the given surface which match the given bound state. Moreover the The "filtered" data is not expected to be set, but the outlier flag should be set for states that are considered to be outlier.
Parameters
geoContextThe current geometry context
calibrationContextpointer to the current calibration context
surfacethe surface at which new track states are to be created
boundStatethe current bound state of the trajectory
prevTipIndex pointing at previous trajectory state (i.e. tip)
trackStateCandidatesa temporary buffer that can be used to collect track states
trajectorythe trajectory to which the new states are to be added
loggera logger for messages
Returns
indices of new track states which extend the trajectory given by prevTip
